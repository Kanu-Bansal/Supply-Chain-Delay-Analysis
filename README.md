# 📦 Supply Chain Delay Analysis & Prediction

## 📊 Project Overview
This project analyzes delivery delays in an e-commerce supply chain and predicts high-risk shipments using machine learning.

## 🛠 Tools Used
- Python (Pandas, Scikit-learn)
- Power BI

## 🔍 Key Features
- Delivery delay analysis
- Vendor performance evaluation
- Risk scoring system
- Delay prediction using Logistic Regression

## 📈 Dashboard Insights
- On-time vs delayed deliveries
- Top delay locations
- Vendor ranking
- Risk distribution

## ⚠️ Unique Feature
- Built a **Risk Score model** to identify high-risk shipments

## 🚀 How to Run
1. Run `Supply Chain Delay Code.py`
2. Open `"Supply_Chain_Delay_Prediction_Dashboard.pbix"` in Power BI
